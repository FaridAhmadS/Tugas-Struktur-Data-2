# Tugas 5 Soal Struktur Data

Project ini berisi beberapa latihan dasar Python menggunakan konsep:
- List
- Set
- Dictionary
- Perulangan
- Function

## 📌 Daftar Program

### 1️⃣ Deduplicasi
Menghapus elemen duplikat dari list tanpa mengubah urutan awal.

File: `deduplikasi.py`

---

### 2️⃣ Intersection Dua Array
Mencari elemen yang sama dari dua list.

File: `intersection.py`

---

### 3️⃣ Anagram Check
Mengecek apakah dua string merupakan anagram.

File: `anagram.py`

---

### 4️⃣ First Recurring Character
Mencari karakter pertama yang muncul lebih dari satu kali dalam string.

File: `first_recurring.py`

---

### 5️⃣ Simulasi Buku Telepon
Program menu sederhana untuk:
- Menambah kontak
- Mencari kontak
- Menampilkan semua kontak

File: `buku_telepon.py`
